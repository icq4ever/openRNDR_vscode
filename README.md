# openrndr_vscode

vscode tasks.json, shel-script for [OPERNDR](https://openrndr.org/)<br/>

## how to use

- put shell script into cloned openrndr template location
- open with vscode
- open command palette and type "task: Run Task" -> OPENRNDR RUN
